Product Quantization (PQ)

Product Quantization is a way to compress vectors, allowing users to save on memory requirements.

  1. query from the user converted into a vector
  2. rough search to gather compressed object vectors that are relevant to the query vector
  3. fetch the uncompressed representations from relevant vectors
  4. rescore the vectors
